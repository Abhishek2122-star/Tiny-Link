## Neon Console: Viewing Your Data

To see your data in real-time:

1. **Go to:** https://console.neon.tech
2. **Select project:** tinylink
3. **Click:** SQL Editor (left sidebar)
4. **Run this query:**

```sql
-- See all links
SELECT * FROM links;

-- See links sorted by most clicks
SELECT * FROM links ORDER BY total_clicks DESC;

-- See recently created links
SELECT * FROM links ORDER BY created_at DESC;

-- Count total links
SELECT COUNT(*) as total_links FROM links;

-- See average clicks per link
SELECT AVG(total_clicks) as avg_clicks FROM links;
```
